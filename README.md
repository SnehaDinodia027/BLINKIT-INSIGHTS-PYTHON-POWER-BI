BLINKIT-INSIGHTS-PYTHON-POWER-BI📊
"End-to-end sales analysis of Blinkit using Python for EDA and Power BI for interactive dashboarding."


🔹 PROJECT FILES
- `Blinkit_Analysis in py.ipynb` → Exploratory Data Analysis (EDA) using Python (Pandas, Matplotlib, Seaborn etc.)
- `BLINKIT DASHBOARD.pbix` → Interactive Power BI dashboard for visualization.



🔹 OBJECTIVES
- Perform data cleaning and exploratory analysis.
- Analyze sales trends and customer behavior.
- Identify top-performing products & categories.
- Build an interactive Power BI dashboard to present insights.



🔹 TOOLS & TECHNOLOGIES
- **Python** (Pandas, NumPy, Matplotlib, Seaborn)
- **Power BI Desktop**
- **Jupyter Notebook**
